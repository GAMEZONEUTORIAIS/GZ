DOWNLOAD

O pacote "All in One Runtimes" reúne todos os pacotes de tempo de execução importantes necessários para garantir que todos os programas funcionem sem problemas e os instale automaticamente também. O pacote de tempo de execução da AiO oferece um instalador conveniente que oferece a opção de selecionar quais tempos de execução a serem instalados! O instalador também detecta tempos de execução já instalados para evitar a reinstalação. As dependências do sistema são verificadas automaticamente. Por exemplo, o .NET Framework 4.6 será instalado no Windows 7 e Windows 8.1 so que no Windows 10 não será.

REQUISITOS MINIMOS
